CSS (Cascading Stylesheets)
- Cascading : According to AI, agar ek hi element par multiple CSS rules lag rahi hain, to browser decide karta hai kaunsa rule zyada important hai aur wahi apply hota hai.
(apni marzi se specific property ko manipulate kar sakte hai, toh utni cheez overwrite hoti hai & baaki inherit hojaati hai)

Browser Default Style = User Agent Style

Ways to write CSS :
1. Inline CSS
- Jaha par humara HTML tag wahi par CSS likhdo
- <h1 style="background-color: #3a3a3a; color:#fff">Chai Aur CSS</h1>

2. Internal CSS:
- CSS is written inside the same HTML file but separately
- Eg :
<head>
<style>
  h1{
    background-color: #3a3a3a; 
    color:#fff
  }
</style>
</head>

3. External CSS
- Written in a separate file & linked to the html file via link tag
- Eg:
  h1{
    background-color: #3a3a3a; 
    color:#fff
  } 

Cascading depends on Line & specificity
Priority : Inline > ID > Class > Tag
